# Tutorial 5: Decoder Plugins

stabstream ships three decoder backends and a plugin interface to add more.
This tutorial covers the Rust-native MWPM decoder (Fusion Blossom) and the
Python adapters for PyMatching, Chromobius, and Tesseract.

---

## Decoder comparison

| Decoder | Language | Algorithm | Best for | p_L quality |
|---------|----------|-----------|----------|-------------|
| `UnionFindDecoder` | Rust | Union-Find O(n·α(n)) | Real-time (< 1 µs) | Near-optimal |
| `FusionBlossomDecoder` | Rust | Fusion Blossom MWPM | Offline research | Optimal (MWPM) |
| `PyMatchingDecoder` | Python | MWPM (C++ core) | Offline research | Optimal (MWPM) |
| `ChromobiusDecoder` | Python | Color-code MWPM | 2D color codes | Optimal |
| `TesseractDecoder` | Python | BP + OSD MLE | qLDPC (BB codes) | Near-ML |

## Rust: FusionBlossomDecoder

Fusion Blossom (Higgott & Gidney 2023) achieves MWPM-optimal logical error
rates with near-linear runtime. Suitable for offline threshold analysis and
as a gold-standard comparison against Union-Find.

### Enabling the feature

Add to your `Cargo.toml`:

```toml
[dependencies]
stabstream-decoder = { path = "...", features = ["mwpm"] }
```

Or for the workspace:

```toml
stabstream-decoder = { path = "crates/stabstream-decoder", features = ["mwpm"] }
```

### Usage

```rust
use std::sync::Arc;
use stabstream_decoder::mwpm::FusionBlossomDecoder;
use stabstream_decoder::Decoder;
use stabstream_dem::{DetectorErrorModel, SpacetimeGraph};
use stabstream_metrics::LogicalErrorAccumulator;

let dem = DetectorErrorModel::parse(&std::fs::read_to_string("circuit.dem")?)?;
let graph = Arc::new(SpacetimeGraph::from_dem(&dem));
let decoder = FusionBlossomDecoder::new(Arc::clone(&graph));
let acc = LogicalErrorAccumulator::new(dem.observable_count);

for window in syndrome_windows {
    let result = decoder.decode_window(&window);
    acc.record(&result, ground_truth_bitmask);
}

println!("p_L = {:.4e}", acc.mean_logical_error_rate());
```

### Threshold sweep with Fusion Blossom

`stabstream-threshold` currently uses Union-Find. To run with Fusion Blossom,
build your own sweep loop:

```rust
use stabstream_sim::DemSampler;
use stabstream_decoder::mwpm::FusionBlossomDecoder;
use rayon::prelude::*;
use rand::{SeedableRng, rngs::SmallRng};

let sampler = Arc::new(DemSampler::from_dem(&dem));
let graph = Arc::new(SpacetimeGraph::from_dem(&dem));
let acc = Arc::new(LogicalErrorAccumulator::new(1));

(0..100_000u64).into_par_iter().map_init(
    || (SmallRng::from_entropy(), FusionBlossomDecoder::new(Arc::clone(&graph))),
    |(rng, decoder), _| {
        let shot = sampler.sample(rng);
        let window = single_round_window(&shot.detector_events, dem.detector_count);
        let result = decoder.decode_window(&window);
        acc.record(&result, shot.observable_flips);
    },
).for_each(|()| {});
```

## Python: PyMatchingDecoder

PyMatching v2 uses a C++ MWPM core with a Python API. Supports both
single-shot and batch decoding.

```bash
pip install pymatching
```

```python
from stabstream import DetectorErrorModel, SyndromeWindow, StabstreamStream
from stabstream.decoders import PyMatchingDecoder
from stabstream import LogicalErrorAccumulator
import numpy as np

dem = DetectorErrorModel.from_file("circuit.dem")
decoder = PyMatchingDecoder(dem)
acc = LogicalErrorAccumulator(dem.observable_count)
window = None

with StabstreamStream("recording.qssf") as stream:
    for frame in stream:
        if window is None:
            window = SyndromeWindow(frame.ancilla_count, window_depth=5)
        window.push(frame)
        if not window.is_full():
            continue

        result = decoder.decode(window.to_numpy_matrix())
        ground_truth = frame.observable_flips or 0

        # Build a minimal result object the accumulator understands
        class _R:
            observable_flips = result["observable_flips"]
            confidence = 1.0
            corrections = []

        acc.record(_R(), ground_truth)

print(f"p_L = {acc.mean_logical_error_rate():.4e}")
```

### Batch decoding (faster for large datasets)

```python
from stabstream.io import load_qssf_batch

decoder = PyMatchingDecoder(dem)
for batch in load_qssf_batch("recording.qssf", batch_size=1024):
    # batch.shape == (n, ancilla_count), dtype=bool
    results = decoder.decode_batch(batch)
    for r in results:
        print(r["observable_flips"])
```

## Python: ChromobiusDecoder

Chromobius wraps PyMatching with a color-code-specific decomposition. It
only supports 2D triangular color codes generated by Stim circuits.

```bash
pip install chromobius stim
```

```python
from stabstream.decoders import ChromobiusDecoder

decoder = ChromobiusDecoder(circuit_file="color_code_d5.stim")
result = decoder.decode(detector_events)
print(f"observable_flips = {result['observable_flips']:#b}")
```

For surface or repetition codes, use `PyMatchingDecoder` instead.

## Python: TesseractDecoder

Tesseract (Google QAI) is a maximum-likelihood decoder for qLDPC codes
using belief propagation + ordered statistics decoding (OSD). It is the
recommended decoder for IBM Bivariate Bicycle codes where MWPM is
suboptimal due to the high-rate, low-distance regime.

```bash
pip install tesseract-decoder   # when publicly released
```

```python
from stabstream.decoders import TesseractDecoder

# Python package mode (when installed)
decoder = TesseractDecoder(dem_file="bivariate_bicycle_144_12_12.dem")

# Subprocess mode (when only the binary is available)
decoder = TesseractDecoder(
    dem_file="bivariate_bicycle_144_12_12.dem",
    binary="/usr/local/bin/tesseract",
)

result = decoder.decode(detector_events)
print(f"observable_flips = {result['observable_flips']:#b}")
```

## Comparing decoders

The `stabstream-threshold` tool uses Union-Find by default. To compare with
MWPM, collect JSON outputs from separate runs and use `compare`:

```bash
# Union-Find sweep
stabstream-threshold run \
    --dem surface_d3.dem --distance 3 \
    --dem surface_d5.dem --distance 5 \
    --p-physical 0.003 --p-physical 0.003 \
    --shots 100000 --decoder union-find \
    --out uf.json

# Plot
stabstream-threshold compare \
    --input uf.json --label "Union-Find" \
    --plot comparison.svg
```

For MWPM, implement the parallel loop shown in the Fusion Blossom section
above and write the output JSON in `RunOutput` format for `compare`.

## Writing your own decoder

Implement the `Decoder` trait in Rust:

```rust
use stabstream_core::window::SyndromeWindow;
use stabstream_decoder::{Decoder, DecoderResult};

pub struct MyDecoder { /* ... */ }

impl Decoder for MyDecoder {
    fn decode_window(&self, window: &SyndromeWindow) -> DecoderResult {
        let active = window.active_detectors();
        // ... your matching / inference logic ...
        DecoderResult {
            corrections: vec![],
            confidence: 1.0,
            observable_flips: 0b01,  // predicted flip of observable 0
        }
    }
}
```

Any type implementing `Decoder` can be passed to `StreamPlayer::analyze()`
and `LogicalErrorAccumulator::record()`.
